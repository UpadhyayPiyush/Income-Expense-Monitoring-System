# 💰 Income-Expense Monitoring System

## Project Overview 
This project is an interactive Personal Finance Tracker Dashboard built using Microsoft Excel.
The dashboard helps users monitor their income, expenses, and savings by transforming raw transaction data into meaningful financial insights through visualization and automated calculations.  

The goal of this project is to demonstrate how Excel can be used as a powerful analytics and reporting tool, not just a spreadsheet application.

## Business Problem 
Many individuals track their daily expenses manually but struggle to understand:  
- Where their money is actually being spent
- Monthly spending patterns
- Whether they are saving enough
Raw transaction data alone does not provide clear financial insights. A structured and visual system is required to simplify financial decision-making.

## Objective 
- Organize and clean financial transaction data
- Categorize income and expenses
- Analyze monthly spending trends
- Calculate total savings and net balance
- Provide an easy-to-understand interactive dashboard

## Dataset Overview 
The dataset contains personal transaction records including:
- Date of transaction
- Income/Expense type
- Category & Sub-category
- Amount
Additional helper columns were created such as:
- Month
- Month Number
- Net Amount (Income – Expense)

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Interactive Filters)
- Conditional Formatting
- KPI Cards
- Data Cleaning & Preparation

- Clear identification of high-spending categories
- Monthly spending trends and patterns

## Recommendations
- Reduce spending in high-expense categories
- Monitor monthly expenses regularly
- Set monthly budget targets
- Use dashboard insights for better financial planning

# Conclusion
This project shows how Excel can function as a data analytics and visualization tool.  
By converting raw financial records into an interactive dashboard, users can easily track their financial health, control spending, and improve savings decisions.
